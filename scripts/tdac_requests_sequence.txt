Additional Notes:
- All requests use Content-Type: application/json
- submitId parameter is unique per session and changes with each request
- hiddenToken is generated after form data is saved and used for subsequent steps
- IDs like nationalityId, country codes, etc. are encrypted/hashed values
- The sequence must be followed in order as later requests depend on data from earlier ones
- For testing, you may need to obtain fresh submitId and tokens from the actual TDAC website
- Some search endpoints may not be strictly required if you have the IDs from previous knowledge
