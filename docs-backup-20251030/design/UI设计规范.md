# 智能出入境助手 - UI设计规范

> **设计理念**: 简洁、高效、适合中老年人使用

---

## 一、设计原则

### 1.1 核心原则

```
1. 简洁至上
   - 每个页面只做一件事
   - 大按钮，大字体
   - 减少选择，明确路径

2. 快速高效
   - 3步完成核心功能
   - 扫一扫 → 选目的地 → 生成完成

3. 老少皆宜
   - 字体大（最小16px）
   - 对比度高
   - 图标+文字双重提示

4. 微信风格
   - 符合微信用户习惯
   - 绿色主色调
   - 熟悉的交互方式
```

### 1.2 目标用户

```
主要用户: 中老年旅行者（45-70岁）
次要用户: 年轻商务旅客（25-45岁）

用户特点:
- 可能不熟悉复杂操作
- 需要清晰的视觉引导
- 喜欢熟悉的交互方式（微信）
- 关注核心功能（快速生成）
```

## 二、设计系统

### 2.1 颜色规范

```
主色（Primary）:
#07C160  微信绿
用途: 主按钮、重要强调、品牌色

辅助色（Secondary）:
#576B95  微信蓝
用途: 链接、次要信息

成功色:
#07C160  绿色
用途: 成功提示、完成状态

警告色:
#FA9D3B  橙色
用途: 警告提示、需要注意

错误色:
#F56C6C  红色
用途: 错误提示、删除操作

背景色:
#FFFFFF  纯白
#F5F5F5  浅灰（背景）
#EDEDED  中灰（分割线）

文字色:
#000000  主要文字（90%不透明度 = #1A1A1A）
#666666  次要文字
#999999  辅助文字
#CCCCCC  禁用文字
```

### 2.2 字体规范

```
字体家族:
iOS: SF Pro Text, PingFang SC
Android: Roboto, Noto Sans SC

字体大小:
H1 - 28px  Bold    页面标题
H2 - 24px  Bold    模块标题
H3 - 20px  SemiBold 小标题
Body1 - 16px Regular 正文（最小）
Body2 - 18px Regular 强调正文
Caption - 14px Regular 辅助说明
Button - 18px SemiBold 按钮文字

行高:
标题: 1.2
正文: 1.5
辅助: 1.4

最小可点击区域: 44x44px（iOS规范）
```

### 2.3 间距规范

```
使用8px网格系统:

基础间距:
4px  - 超小间距（图标与文字）
8px  - 小间距（相关元素）
16px - 标准间距（页面边距）
24px - 中等间距（模块之间）
32px - 大间距（页面分组）
48px - 超大间距（页面区块）

页面边距: 16px（左右）
卡片圆角: 12px
按钮圆角: 8px（小按钮）/ 24px（大按钮）
输入框圆角: 8px
```

### 2.4 组件规范

#### 按钮

```
主按钮（Primary Button）:
- 背景: #07C160
- 文字: #FFFFFF, 18px, SemiBold
- 高度: 48px
- 圆角: 24px
- 阴影: 0 2px 8px rgba(7, 193, 96, 0.3)
- 按下: 背景变为 #06AD56

次要按钮（Secondary Button）:
- 背景: #FFFFFF
- 边框: 1px #07C160
- 文字: #07C160, 18px, SemiBold
- 高度: 48px
- 圆角: 24px

文字按钮（Text Button）:
- 背景: 透明
- 文字: #576B95, 16px, Regular
- 下划线: 无
```

#### 输入框

```
标准输入框:
- 背景: #F5F5F5
- 边框: 1px #EDEDED
- 圆角: 8px
- 高度: 48px
- 内边距: 12px 16px
- 占位符: #CCCCCC, 16px
- 文字: #1A1A1A, 16px
- 焦点: 边框变为 #07C160

错误状态:
- 边框: 1px #F56C6C
- 底部提示: #F56C6C, 14px
```

#### 卡片

```
标准卡片:
- 背景: #FFFFFF
- 圆角: 12px
- 阴影: 0 2px 12px rgba(0, 0, 0, 0.08)
- 内边距: 16px

可点击卡片:
- 按下: 背景变为 #F5F5F5
- 右侧箭头: > (#CCCCCC)
```

## 六、交互设计

### 6.1 手势

```
点击（Tap）:
- 标准点击
- 反馈: 0.1秒高亮
- 适用: 按钮、卡片、列表项

长按（Long Press）:
- 按住0.5秒
- 反馈: 振动+菜单
- 适用: 证件卡片（显示编辑/删除）

滑动（Swipe）:
- 左滑删除
- 右滑返回（iOS）
- 适用: 列表项

下拉刷新（Pull to Refresh）:
- 下拉loading动画
- 释放触发刷新
- 适用: 历史列表
```

### 6.2 动画

```
页面转场:
- 推入/推出（Push/Pop）
- 时长: 300ms
- 缓动: ease-in-out

卡片展开:
- 缩放+淡入
- 时长: 200ms
- 适用: 弹窗、详情页

加载动画:
- 旋转或脉冲
- 循环播放
- 使用Lottie动画

成功动画:
- 绿色勾放大
- 时长: 500ms
- 带弹性效果
```

### 6.3 反馈

```
视觉反馈:
- 按钮按下: 背景色变深10%
- 卡片点击: 高亮0.1秒
- 输入框焦点: 绿色边框

触觉反馈（Haptic）:
- 轻触: 按钮点击
- 警告: 错误提示
- 成功: 操作完成
- 仅iOS支持

音效反馈（可选）:
- 点击: 轻微咔哒声
- 成功: 悦耳提示音
- 错误: 警告音
- 默认关闭
```
