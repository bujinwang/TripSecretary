## Interaction Patterns

### Message Sending
1. User types message
2. Show typing indicator
3. Disable input temporarily
4. Display user message
5. Show AI typing indicator
6. Display AI response
7. Enable input
8. Show quick replies (if any)

### Quick Reply Selection
1. User taps quick reply button
2. Button briefly highlights
3. Message auto-sends
4. Quick replies disappear
5. AI response appears

### Flight Selection
1. User taps flight card
2. Card expands with details
3. Show "Select" button
4. Confirm selection
5. Add to trip plan
6. Return to chat with confirmation

## Accessibility

- VoiceOver support for all text
- Minimum touch target: 44x44pt
- High contrast mode support
- Dynamic font sizing
- Descriptive labels for icons
- Keyboard navigation support
